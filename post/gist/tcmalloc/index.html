libunwind

libunwind库为基于64位CPU和操作系统的程序提供了基本的堆栈辗转开解功能，其中包括用于输出堆栈跟踪的API、用于以编程方式辗转开解堆栈的API以及支持C++异常处理机制的API。

# 64位操作系统安装 libunwind库.32位不要安装
yum install libunwind-devel

# 或者编译安装
git clone git://git.sv.gnu.org/libunwind.git
cd libunwind
./autogen.sh
./configure
make &&make install
